# Nexustal (Community Edition) 🚀

Nexustal is an ultra-fast, open-source, and self-hosted API engineering and documentation platform designed for modern development teams. Built with a high-performance **C++ backend core** and a reactive **Nuxt/Vue.js frontend**, Nexustal provides an ecosystem that replicates the core features of Postman and Insomnia while keeping your sensitive API data 100% on your own infrastructure.

---

## ✨ Key Features

- **Blazing Fast Performance:** Powered by a native C++ backend ensuring minimal memory footprint and sub-millisecond response processing.
- **Self-Hosted & Privacy-First:** Free forever for self-hosting. Deploy via Docker with absolute ownership of your team's API data.
- **Multi-Protocol Support:** Design, document, and test RESTful, GraphQL, SOAP, and WebSocket APIs in one unified workspace.
- **Granular RBAC (Role-Based Access Control):**
  - **Owner:** Full project governance.
  - **Backend Developer:** Full CRUD operations on endpoints, release publishing, and visibility toggles.
  - **Frontend Developer:** Live documentation viewing, API execution/testing, commenting with `@mentions`, and creating API requirement drafts.
  - **Viewer:** Read-only access with commenting privileges.
- **Real-Time Collaboration:** Powered by Server-Sent Events (SSE) for instant in-app notifications and real-time team synchronization.
- **Advanced API Testing Suite:** Embeds an isolated JavaScript sandbox (via QuickJS) to execute Pre-request scripts and Post-response validation assertions (API Chaining).
